Introduction
eDuck is an academic information system whose main role is to help organize university studies - namely courses. It was created as a final project by D. Kamenská, V. Borodin and T. Peregrín - three students from Matej Bel University in Banská Bystrica.
This guide will take you through the many useful features and functions of eDuck and will help you get better acquainted with the system.
Notable features:
  • Homepage
  • Courses
  • User Management
  • Account Management

Courses
A User has access to the whole list of courses with all the important information listed. To make manipulation with the list easier, a search bar is available to find the desired course by its title, as well as many filters that let you filter through the list in an ascending or descending order. A specific feature only available for Admins, Referents or Teachers is the ability to create a new course which can be accessed by clicking the New course button which then opens up a short form.
A useful feature is the ability to export the whole list or just some of the courses into a pdf. This can be done by clicking the checkbox next to the listed course or by clicking the main one in the head of the table. Such an action let's two separate buttons appear - Export button which, as mentioned before, let's you export the table. While the other button - Remove is only visible for Admins and gives you the option to delete the desired course or courses.
Clicking on any course listed in the table takes you to a page with more detailed information about the course and gives you different options depending on your role. Admin, Referent or a Teacher is able to edit the chosen course, while a Student has the ability to sign up for the chosen course, but only in the specified sign up period.
User Management
This section of the manual is mostly about user management which is a specific feature that only someone with a role of Admin has access to. If you're looking for information on how to change your password or username, please refer to the Account Management section.
Creating a new user is fairly simple. Clicking on the Plus sign which is located in the Accounts section of the sidebar takes you to the user creation page where you have the option to create 4 different types of users - Manager/Admin, Referent, Teacher or Student. After choosing the desired type of user, all you need to do is fill out the form with the correct information about the user and click the Create button.
All the sections listed under the Accounts section lead to lists of users - specifically lists of Teachers, Students and Referents. To make manipulation with the list easier, a search bar is available to find the desired user by their last name, as well as many filters that let you filter through the list in an ascending or descending order. A New user button takes you back to the user creation page where you can create a new user.
You also have the ability to export the whole list or just some of the users into a pdf. This can be done by clicking the checkbox next to the listed user or by clicking the main one in the head of the table. Such an action let's two separate buttons appear - Export button which, as mentioned before, let's you export the table. While the other button - Remove gives you the option to delete the desired course or courses.
